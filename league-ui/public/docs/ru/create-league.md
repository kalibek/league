# Создание и ведение лиги

Это пошаговое руководство по полному циклу работы с лигой — от создания до проведения регулярных ежемесячных мероприятий.

## Шаг 1 — Войти в аккаунт

Для создания лиги и управления ею необходимо быть авторизованным. Используйте email и пароль или аккаунт Google на странице входа.

## Шаг 2 — Создать лигу

Нажмите кнопку **Создать лигу** на главной странице. Заполните:

- **Название** — название вашей лиги
- **Описание** — дополнительные сведения о лиге (необязательно)

## Шаг 3 — Настроить лигу

Задайте параметры, которые будут применяться ко всем мероприятиям лиги:

| Параметр | Описание |
|---|---|
| **Размер группы** | Количество игроков в одной группе |
| **Игр для победы** | Количество партий для победы в матче (например, до 3 побед в серии до 5) |
| **Количество повышений** | Сколько лучших игроков переходят в группу выше |
| **Количество понижений** | Сколько худших игроков переходят в группу ниже |
| **Количество столов** | Доступные столы (используются при составлении расписания) |

## Шаг 4 — Создать первое мероприятие

После сохранения лиги создайте первое **мероприятие**, указав:

- **Дату начала**
- **Дату окончания**

Одно мероприятие соответствует одному месячному циклу игр.

## Шаг 5 — Добавить игроков в группы

Распределите игроков по группам вручную или воспользуйтесь инструментом **жеребьёвки**, который распределяет игроков автоматически на основе их рейтинга. Каждая группа (S, A1, A2, B1, B2, ...) должна быть заполнена перед генерацией расписания.

## Шаг 6 — Сгенерировать расписание кругового турнира

После заполнения групп создайте расписание матчей для каждой группы. Приложение формирует полный круговой турнир — каждый игрок встречается с каждым ровно по одному разу.

## Шаг 7 — Провести мероприятие

В ходе мероприятия:

- **Вводите счёт** после каждого матча. Результаты обновляются в реальном времени для всех зрителей.
- **Отмечайте DNS** для игроков, которые не явились. Неявившиеся исключаются из итоговой таблицы и автоматически понижаются.

## Шаг 8 — Завершить группы и мероприятие

Когда все матчи в группе сыграны, отметьте группу как **завершённую**. После завершения всех групп отметьте **мероприятие как завершённое**. Приложение рассчитает итоговые позиции, обновит рейтинги Glicko2 и определит повышения и понижения.

## Шаг 9 — Создать следующее мероприятие

Используйте действие **Создать следующий черновик**, чтобы автоматически распределить игроков по группам на следующий месяц с учётом всех переходов из завершённого мероприятия. Затем повторите с шага 4.
