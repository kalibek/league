# Топтар арасындағы ауысулар

## Дивизиондар құрылымы

Ойыншылар топтар иерархиясына бөлінеді. Ең жоғары топ — **S** (суперлига), одан кейін **A1**, **A2**, **B1**, **B2** және т.б. Әрбір топ іс-шара шеңберінде өз бетінше ойнайды, бірақ нәтижелер ойыншылардың келесі айдағы ауысуын анықтайды.

```
S  ←→  A1  ←→  A2  ←→  B1  ←→  B2  ←→  ...
```

A1-ден жоғарылаған ойыншылар S тобындағы орындарды толтырады; S тобынан түсірілгендер A1-ге өтеді. Тізбек барлық дивизиондар бойынша жалғасады.

## Жоғарылау және төмендеу

Әрбір топ үшін екі сан белгіленеді:

- **Жоғарылаулар саны** — жоғары топқа өтетін үздік ойыншылар саны
- **Төмендеулер саны** — төмен топқа өтетін соңғы ойыншылар саны

Бұл параметрлерді лига әкімшісі құру кезінде белгілейді; әртүрлі топтар үшін мәндер әр басқа болуы мүмкін.

## DNS (Келмеген ойыншы)

**DNS** (Did Not Show) белгісі қойылған ойыншы іс-шараға келмеген. Мұндай ойыншылар кестедегі орнына қарамастан әрқашан төмендейді.

DNS ойыншылары төмендеу слоттарын мынадай ережеге сәйкес алады:

```
delta = ТөмендеулерСаны − dnsСаны
```

- **delta > 0** болса: соңғы `delta` белсенді (DNS емес) ойыншылар да төмендейді.
- **delta ≤ 0** болса: барлық төмендеу слоттары DNS ойыншыларымен толтырылған; белсенді ойыншылар төмендемейді.

**Мысал:** Топта төмендеулер саны = 2, бір DNS ойыншысы бар.
- delta = 2 − 1 = 1
- DNS ойыншысы төмендейді, плюс 1 соңғы белсенді ойыншы да төмендейді.

**Мысал:** Топта төмендеулер саны = 2, үш DNS ойыншысы бар.
- delta = 2 − 3 = −1 (≤ 0)
- Үш DNS ойыншысы да төмендейді, белсенді ойыншылар төмендемейді.

## Тең орындарды шешу

Екі немесе одан да көп ойыншы бірдей жеңіс санына ие болса, мына кезектілік қолданылады:

1. **Негізгі ұпайлар** — топтың барлық матчтарындағы жиналған ұпайлар жиынтығы.
2. **Тайбрейк ұпайлары** — тек тең тұрған ойыншылар арасындағы матчтардан есептелген ойын айырмасы (жеңілген минус ұтылған).
3. **Жеңіс/жеңіліс арақатынасы** — топтың барлық матчтарындағы жеңістерді жеңіліске бөлу.
4. **Бас судья шешімі** — 1–3 қадамдарынан кейін де тең болса, орынды бас судья қолмен анықтайды.

### Нақты мысал

Үш ойыншы — Айгүл, Бекзат және Гүлнар — 2 жеңіс, 1 жеңіліспен аяқтайды.

- **1-қадам:** Үшеуінде де 6 ұпай. Теңдік сақталады.
- **2-қадам:** Айгүл–Бекзат, Айгүл–Гүлнар, Бекзат–Гүлнар матчтары бойынша:
  - Айгүл: ойын айырмасы +2
  - Бекзат: ойын айырмасы 0
  - Гүлнар: ойын айырмасы −2
  - Гүлнар тайбрейктен шығарылады. Айгүл Бекзаттан жоғары орын алады.
- Қорытынды тәртіп: **Айгүл → Бекзат → Гүлнар**.

2-қадам да тең болған жағдайда 3-қадам (жеңіс/жеңіліс арақатынасы) тексерілер еді, одан кейін 4-қадам.
